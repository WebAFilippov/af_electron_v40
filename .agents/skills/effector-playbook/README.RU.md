# Effector Playbook Skill

Скилл с короткими правилами для написания корректного и надежного кода на Effector и исправления частых ошибок. Заточен под LLM и акцентирует явное управление потоками, тестируемость, SSR‑безопасность и правильное моделирование.

## Что внутри
- Набор коротких правил с примерами (`rules/`)
- Скомпилированный список правил (`AGENTS.md`)
- Полная API‑выгрузка Effector (`references/llms-full.txt`)
- Отдельные справки по редким темам (SSR/scopes, тестирование, Babel/SWC plugin)
- Подсказки по Patronum и нюансы версий

## Быстрый старт

Установить одной командой:

```bash
npx skills add neolite/effector-playbook
```

## Структура
- `SKILL.md` — метаданные и индекс правил
- `AGENTS.md` — все правила в одном файле
- `rules/` — отдельные правила по категориям
- `references/` — справочные материалы

## Установка

Используйте skill‑менеджер (например, skill.sh) и добавьте этот репозиторий, или установите напрямую:

```bash
npx skills add neolite/effector-playbook
```

## Использование
1. Откройте `SKILL.md` и найдите нужные правила.
2. Читайте конкретные файлы в `rules/`.
3. Для редких кейсов используйте материалы из `references/`.

## Вклад
- Держите правила короткими и конкретными.
- Лучше примеры кода, чем длинные объяснения.
- Не дублируйте одно и то же в правилах и справках.
